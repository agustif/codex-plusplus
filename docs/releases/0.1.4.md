# Codex++ 0.1.4

Codex++ 0.1.4 is an `agustif` fork release. It keeps the upstream 0.1.3 behavior while moving the install, self-update, issue-reporting, and default tweak download paths onto the `agustif` fork family.

## Changed

- `install.sh`, `install.ps1`, `update.sh`, and `update.ps1` now default to `agustif/codex-plusplus`.
- `codexplusplus update` and `codexplusplus self-update` now default to `agustif/codex-plusplus`.
- Runtime update checks and in-app bug report links now point at `agustif/codex-plusplus`.
- Default tweaks now install from `agustif/codex-plusplus-keyboard-shortcuts` and `agustif/codex-plusplus-bennett-ui`.

## Validation

- Built with `npm run build`.
- Tested with `npm test`.
